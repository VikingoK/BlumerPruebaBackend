# Instrucciones

### Las librerias instaladas son las siguientes:

pip install fastapi uvicorn sqlalchemy psycopg2

pip install 'strawberry-graphql[fastapi]'

## Arrancar proyecto

uvicorn main:app --reload

## Ruta acceso

http://localhost:8000/graphql

## REVISAR

En los archivos hay una carpeta llamada recursos, ahi se encuntran ubicados el script de la base datos postgresql y se encuntran las mutaciones y querys pedidos
